# SofiaTalks-Partnership-Proposal
Partnership bussiness deck created for **SofiaTalks**, designed to present partnership opportunities and drive partner purchases for one of Bulgaria’s leading youth career and entrepreneurship events

## About

This partnership deck was created for a major youth-focused career and entrepreneurship event in Sofia, Bulgaria.

Its structure and content were shaped by:

- marketing research
- competitor and proposal analysis
- UX-driven structuring
- bilingual presentation for wider usability in different partnership contexts

The final result is a proposal designed not only to communicate the value of the event, but also to convert interest into actual sponsorship engagement.

## Impact

This concrete sponsorship deck successfully helped secure **almost all event partners**, making it a key part of the project’s partnership and outreach process.
